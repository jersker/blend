## To Use:

### Search Tab
Do one of the following:

- Enter a 3 or 6 character hexadecimal value (charactera 0-9 and A-F).
- Enter individual red, green, and blue values (integers 0-255).

#### Valid Inputs
`DAD`
`DAD357`
`255`, `255`, `255`
