Valid Color Codes

  • Hexadecimal Colors
  • Hash symbol # optional
  • Use letters A-F and integers 0-9
  • Must have either 3 or 6 characters following the optional hash symbol
  • RGB Colors
  • RGB prefix RGB(...) optional
  • Use integers 0-255
  • Separate values by commas, spaces, or both
